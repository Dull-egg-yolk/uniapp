# 库存管理小程序
## 介绍
该小程序用户库存管理


## 功能
1.录入库存的物品，生成二维码。后续扫码识别物品、存放位置，继而进行入库、出库等操作
2.可便捷的进行盘点，并生成盘点报表
3.可查看物品的出入库明细、实时库存、月度消耗汇总等
4.报表可用邮件等形式发出，整个过程手机操作，不需要电脑

## 使用方法
1. 克隆仓库到本地 git clone [你的项目仓库地址]  cd [项目文件夹名]
2. 安装依赖 npm install
2. 在微信开发者工具中打开项目（unpackage/dist/dev/mp-weixin）
3. 配置小程序的appid （测试：填写「测试号」（微信开发者工具可直接选择）正式：填写小程序官方 AppID）
4. 运行小程序

## 注意事项
- 请确保已安装微信开发者工具
- 请确保已配置小程序的appid
- 请勿直接导入 UniApp 项目根目录（会报 app.json 缺失 错误），必须导入打包后的 mp-weixin 目录
- 若提示「未找到 app.json」，检查打包是否成功，或目录选择是否正确
- 首次导入可能提示「代码未上传」，忽略即可（开发阶段无需上传）
